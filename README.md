## Ubuntu

Install [Android Debug Bridge (adb)](https://developer.android.com/studio/command-line/adb):

```bash
  sudo apt install adb
```

Rename *miui-debloater.cmd* to *miui-debloater.sh* and run it:

```bash
  bash './miui-debloater.sh'
```
